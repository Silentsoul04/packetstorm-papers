HTML Colors

To specify a color in HTML, use the RGB triplet or color name. RGB triplets work in most browsers. So, as an example, to put an Aliceblue background on a page, you'd use this tag:

<BODY BGCOLOR=#F0F8FF>

Use the Decimal number in ActiveX controls and Java applets that require decimal numbers.
