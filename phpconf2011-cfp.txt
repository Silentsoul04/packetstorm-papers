Auf der Zielgeraden zur IPC Spring möchten wir Euch schon jetzt
einladen, Eure Themen, Ideen, Vorschläge für die International PHP
Conference im Oktober einzureichen. Die International PHP Conference
findet vom 9. bis 12. Oktober 2011 in der Rheingoldhalle in Mainz statt
und läuft parallel mit der WebTech Conference.

Wie immer bieten wir auch eine Reihe von Special Days an und stehen Euch
natürlich auch auf der International PHP Conference in Berlin für
Gespräche zur Verfügung.

Wir freuen uns auf eure Vorschläge unter: http://phpconference.com/input

Themen der Konferenz
=> PHP Core
=> PHP & Web Security
=> Web Architecture
=> Best Practices
=> Scaling & Performance
=> Agile Development
=> Continuous Integration
=> Tools & Frameworks
=> Frontend Development
=> Database (NoSQL)
=> Cloud Technology

ALLE INFORMATIONEN IM ÜBERBLICK:

* Datum: 9. - 12. Oktober 2011
* Ort: Rheingoldhalle Mainz
* Einreichen der Vorschläge via http://phpconference.com/input
* Deadline für den Call for Papers: 10. Juni 2011
* Dauer einer Keynote: 45 Minuten
* Dauer einer Session: 60 Minuten
* Dauer der Workshops: 4 x 90 Minuten (Full Day) oder 2 x 90 Minuten
(Half Day)

Wir freuen uns bereits auf viele spannende Vorschläge und Anregungen!

Freundliche Grüße
Euer International PHP Conference Team
